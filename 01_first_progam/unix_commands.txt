1. mkdir <directory_name/path> - (make directory)
2. cd <directory_name/path> - (change directory)
2.1 cd.. - moves to the parent directory
3. ls (list) - Shows the content of current directory
4. code <file_name> - creates a files and opens it in the text editor
5. gcc <file_name.c (source code file)> - compiles the C program. Creates a.out executable file
6. ./a.out - run the executable file
7. pwd (print working directory) shows the path of the current directory
